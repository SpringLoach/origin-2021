#### 原型  
> 通过 `new function(){}` 返回的是一个继承自 Object 的实例对象，没有继承 Function。  

```
Object.prototype.a = function(){};
Function.prototype.b = function(){};
var f = new function(){};
var f1 = new Function

console.log(f.a);   // ƒ (){}
console.log(f.b);   // undefined
console.log(f1.a);  // ƒ (){}
console.log(f1.b);  // ƒ (){}
```
