#### Switch语句  
> 基于不同条件执行不同动作。  

```
switch(表达式) {
     case a:
        代码块
        break;
     case b:
     case c:
        代码块
        break;
     default:
        默认代码块
} 
```

关键词 | 类型 
:-: | :-: 
case  | 与 switch 中的表达式比较。多个 case 匹配时，执行首段代码
break | 跳出 switch 代码块，节省执行时间。没必要中断最后的代码块
default | 不存在 case 匹配时所运行的代码，可以不放最后
